🚀 AS-WEB-HUB | AI Pro Digital Hub

AS-WEB-HUB is a smart web application designed to bring digital services under one roof. It features AS-AI, an intelligent assistant that provides step-by-step guidance by understanding website layouts.
🌐 [മലയാളം വിവരണത്തിന് താഴേക്ക് സ്ക്രോൾ ചെയ്യുക]
✨ Key Features

    AS-AI Expert Guidance: Intelligent AI that locates buttons and menus on websites.

    Service Management: Add, edit, and organize your favorite digital services.

    AI Master Guide: Automatically generate comprehensive usage guides for any service.

    Personal AI Notebook: A dedicated space to get specific answers to your queries.

    GitHub Ready: Easily backup and store your data as JSON.

🛠️ How to Use

    Download the repository or open the live GitHub Pages link.

    Go to the Developer page and enter your Gemini API Key.

    Add your desired services (e.g., KSEB, Akshaya, Passport Seva).

    Click "AI Guide & Notes" to get instant help from AS-AI.

💻 Tech Stack

    Frontend: HTML5, CSS3, JavaScript

    AI Engine: Google Gemini AI (1.5 Flash)

    Icons: Font-Awesome
